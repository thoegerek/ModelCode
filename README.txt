Code here used to generate "Knudsen et al. n.d. Migration through social learning and consequences for population dynamics and evolutionary stability" (unpublished)

To be made more easily accessible and usable after submission.



To create any and all figures in the manuscript, run corresponding script (if not clear, run all) with desired parameters (to be renamed) to generate "data" .mat-files. WARNING: some are very big to ensure high figure fidelity; consider reducing.

Then run "ArticleFigures.m" to generate all article figures (when uncommented)
